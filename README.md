## [HC32F005简介](https://github.com/SoCXin/HC32F005/wiki)

[HC32F005](http://www.ti.com.cn/product/cn/HC32F005) 为Low Pin宽电压工作范围的MCU，12位1Msps采样SARADC内置运放

#### 关键特性

* 宽电压1.8～5.5V
* TSSOP20 引脚兼容
